Favorite Flix

This API, when completed, will provide users with information about their favorite movies, directors and genres. Users will be able to create and customize their profile and create a list of their own personal favorite movies.

URL Endpoints and Request Methods

Request URL HTTP Method Response Data
Get a list of movies /movies GET A JSON object holding data about all the movies
Get a list of all users /users GET A JSON object holding the data of all users
Get data about a movie from the title movies/[Title] GET A JSON object holding the data of the movie
Return data about a genre of movie /movies/Genre/[Name] GET A JSON object holding the data of a genres description and movies of that genre
Return info about a user from their Username /users/[Username] GET A JSON object holding data about a user
Return info about the director by name /movies/Director/[Name] GET A JSON object holding data about a movie containing the director name
Add a new movie title to the list of movies /movies POST A JSON object showing the new movie addedd { Write in this syntax Title: , Description: , Genre : { Name: , Description: }, Director: { Name: , Bio: }, ImagePath: }
Delete a movie from the list /movies/[Title] DELETE To remove a movie from the movies collection using the movie ID. A text message will appear indicating the movie was deleted
Delete a user from the list /users/[Username] DELETE To remove a user from the users collection. A text message indicating the user was deleted
Register a new user /users POST To add a new user to the users collection. A text message indicating the user was created Write in this syntax { Username: , Password: , Email: , Birthday: }
Update a users username /users/[Username] PUT A JSON object containing the updated user data Write in this syntax { Username: , Password: , Email: , Birthday: }
Add a movie to the users fav movies /users/[Username]/movies/[MovieID] POST A text message indicating the movie has been added to the array of their favorite movies.
Update a users username /users/[Username]/movies/[MovieID] DELETE A text message indicating the movie has been removed from the array of their favorite movies
