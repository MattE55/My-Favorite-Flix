Favorite Flix

This API, when completed, will provide users with information about their favorite movies, directors and genres. Users will be able to create and customize their profile and create a list of their own personal favorite movies.

URL Endpoints and Request Methods

Request URL HTTP Method Response Data
Get a list of movies /movies GET A JSON object holding data about all the movies
Get data about a movie from the title movies/[Title] GET A JSON object holding the data of the movie
Return data about a genre of movie /movies/[Genre] GET A JSON object holding the data of a movie containing the genre
Return info about the director by name /movies/[Director] GET A JSON object holding data about a movie containing the director name
Add a new movie title to the list of movies /movies/[Num]/[Title] POST A text message indicating the movie has been added, including the number where it is on the list
Delete a movie from the list /movies/[Title] DELETE A text message indicating the movie was deleted
Delete a user from the list /users/[Username] DELETE A text message indicating the user was deleted
Register a new user /users POST A text message indicating the user was created
Update a users username /users/[Username] PUT A text message indicating the username was uptaded
